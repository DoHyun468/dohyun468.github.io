# dohyun468.github.io

Public portfolio site for Dohyun Kim.

## Local Preview

```bash
python3 -m http.server 8770 --directory .
```

Open:

```text
http://localhost:8770/
```

## GitHub Pages

Create a GitHub repository named:

```text
dohyun468.github.io
```

Push this folder to the repository root. GitHub Pages will serve it at:

```text
https://dohyun468.github.io/
```

## Public / Private Split

This repository is for the public portfolio only.

Do not include:

- private application dashboard
- cover letter drafts
- company-specific strategy notes
- personal phone number
- private JD analysis
